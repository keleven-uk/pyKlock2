About pyKlock2

Klock - a clock with a k.

Written in Python 3.11.1

Klock will consist of a number of views, or will be [similar to lazKlock, klock.vb & pyKlock], at the moment a trying ground for ideas.  Stuff that I come across that might be useful.

pyKlock was written in Tkinter, but using the framework pySimpleGUI.
pyKlock2 is written in pure Tkinter only - hopefully be simpler.
